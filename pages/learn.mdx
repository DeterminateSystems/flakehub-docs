# Learn Nix

[Determinate Systems][detsys] has created a learning resourced called [Zero to Nix][z2n] that we recommend as a guide to getting started with Nix.

<Z2NButton />

Unlike other learning resources in the [Nix] ecosystem

## Start

The main learning track in Zero to Nix is the [Quick start][start], which includes eight practical tutorials:

1. [Get Nix running on your system][start-1]
1. [Run a program with Nix][start-2]
1. [Explore Nix development environments][start-3]
1. [Build a package using Nix][start-4]
1. [Search for Nix packages][start-5]
1. [Turn your project into a flake][start-6]
1. [Uninstall Nix (if necessary)][start-7]
1. [Learn more][start-8]
